# FIGMA DESIGN BRIEF
## Work-Life Balance - Vocation Infographics

**Project:** Create 6 static infographic panels (one per vocation)  
**Purpose:** Replace double-sided vocation cards with single-page reference sheets  
**Format:** Static informational graphics (NO interactive elements, NO buttons, NO UI logic)  
**Target:** Print-ready PDF or digital display

---

## GENERAL DESIGN GUIDELINES (Apply to ALL Vocations)

### 1. Visual Style Consistency
- **Style:** Flat icons, rounded rectangles, soft gradients
- **Avoid:** Realism, heavy textures, 3D effects
- **Color:** Calm, readable color contrasts
- **Typography:** Clear, readable fonts with good hierarchy

### 2. Layout Structure (Universal Template)
Each infographic must follow this three-zone structure:

```
┌─────────────────────────────────────────┐
│  [HEADER - VOCATION IDENTITY]          │
├─────────────────────────────────────────┤
│  [CAREER LEVELS - 3 COLUMNS]            │
│  ┌─────────┬─────────┬─────────┐       │
│  │ Level 1 │ Level 2 │ Level 3 │       │
│  └─────────┴─────────┴─────────┘       │
├─────────────────────────────────────────┤
│  [SPECIAL EVENTS - 2 PANELS]             │
│  ┌──────────────┬──────────────┐        │
│  │ Special      │ Special      │        │
│  │ Event 1      │ Event 2      │        │
│  └──────────────┴──────────────┘        │
└─────────────────────────────────────────┘
```

### 3. Level Column Structure (Common Template)
Each level column must contain, in this exact order:
1. **Level Number** (e.g., "LEVEL 1")
2. **Job Title** (e.g., "Junior Clerk")
3. **Salary** (Icon + Text: "+100 VIN / AP")
4. **Promotion Requirements** (Knowledge, Skills, Experience/Award)
5. **Level Perks** (Always-on effects, bullet list with icons)
6. **Work Obligation Rule** (if applicable, with footnotes)
7. **Level Action** (Dice-based action with outcomes)
8. **Waive Rule** (if applicable)

### 4. Information Design Rules
- **DO NOT** mix information between levels
- **DO NOT** hide penalties in icons only — text is mandatory
- Repeated rules may be visually grouped but must remain readable per level
- **NO** interactive hints, arrows, or buttons
- Must work as a printed PDF

### 5. Color Coding
- Each vocation has its own color scheme
- Use vocation color for: Header bar, section titles, icon backgrounds
- **DO NOT** use colors of other vocations
- Maintain consistent color application across all levels

**Special Event 2 – Forced Protective Removal**

  - **5–6:** Permanent removal: child placed in state care permanently; parent loses **–6 Satisfaction** and no longer pays/spends AP/gains Satisfaction from this child → you gain **+4 Satisfaction**
